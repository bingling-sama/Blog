# 申请 GitHub 学生包踩坑实录
早早听闻 GitHub 的学生福利十分丰厚，我一直都很想申请一份玩玩，于是乎在前面的四个月中，我曾试过用录取通知书、学生证去申请，但都被机申以各种理由拒了。这个月终于能在学信网上查到学籍信息了，于是我用学信网的在线学籍认证成功申请到了 GitHub 学生包，这里记录一下踩坑过程。

## 准备材料
要准备的东西很简单，但都是必须的。

### .edu 邮箱
这一步因学校而异，反正你要有一个由你本人控制的 .edu 邮箱，`.edu` 或者 `.edu.cn` 都可以。


### 学信网学籍认证报告
首先，注册登录学信网并完成实名认证。然后点击导航栏的 `学信档案`，再点击右侧 `在线验证报告`，点击链接进入。这里会要求你登录学信档案，用学信网账号登录就可以。

此时你应该来到了这个界面

![image](https://cdn.ipfsscan.io/weibo/large/007CWdRmgy1icmz2ivsz8j31wm13jb1s.jpg)

一般在校大学生选择第一个报告就好，点击查看，如果你没有申请过报告，会提示你申请。

在线验证报告申请完毕以后，将 pdf 下载下来，然后找一个可以翻译文档的工具将它翻译成英文版，并转换成 png/jpg 格式。

## 填写信息

### 个人信息

首先打开你 GitHub 的 profile，这里有几个字段需要稍微修改一下以符合申请标准。

- Name：填你验证报告上姓名的拼音
- Bio：加一句类似 *Student of xxxx* 的介绍
- Company：填你学校的英文全称或者缩写
- Location：选择你学校所在的地区

### 付款信息

打开你 GitHub 账号的 Settings 页面，点击 `Billing and plans`，再点击 `Paymant information`，点击编辑。

- First name/Last name：如上，填入姓名拼音
- Address：直接填写大学名称或校区地址
- City/Country：同上
- State/Province：有些大学在申请时可能会检查此字段，最好也如实填写

### 绑定邮箱

还是在 Settings 页面，点击 `Email`，将你的 `.edu` 邮箱添加到账号，并查收验证邮件完成验证。

> 这一步非常重要，一定要完成 `.edu` 邮箱的验证，否则可能导致申请被拒。

## 提交申请

验证过 `.edu` 邮箱以后，回到 Dashboard 页，应该能看到 GitHub 邀请你加入 Education 计划的卡片了，可以直接点击申请。

这里建议关掉所有代理软件并且使用校园网访问，保证 IP 地址和定位是在校内。

在申请页，如实填写所有字段，最好全部使用英文及官方译名，在上传凭证时上传刚才准备好的英文版验证报告，Proof Type 选择 Others。

这里可能会要你提供对于证明的描述，用英文写一句 `Official proof of CHSI` 就可以了。

点击提交申请，理论上可以直接通过，如果被 Reject 了 GitHub 会显示原因并给出相应的提示，按照提示修改以后重新提交就可以了。

在申请通过的三天内，经过人工复核，GitHub 会向你的 `.edu` 邮箱发送邮件提醒你你的学生权益已经成功激活，现在你就可以愉快白嫖各种学生福利啦。~~我在正好第三天结束时才收到邮件~~
